## üìñ Project Overview

**TrueTrip** is a smart travel planner that goes beyond simple place listings. It uses AI to curate personalized itineraries based on your taste, age group, and companions, ensuring a trip that is rich in context and feasible in schedule.

### üí° Key Features
1.  **AI Planner:** Input your preferences (Themes, Age, Group Size) and get a tailored 5-spot course generated by `gpt-4o-mini`.
2.  **Explore:** Browse reliable travel spots with historical context and wiki links.
3.  **Route Optimization (Planned):** Realistic scheduling using TSP algorithms.

## üöÄ Getting Started

### 1. Backend (Spring Boot)
Requires JDK 21+.

1.  **Configure API Key:**
    Create `backend/src/main/resources/application-secret.properties`:
    ```properties
    openai.api-key=YOUR_OPENAI_API_KEY
    ```
2.  **Run Server:**
    ```bash
    cd backend
    ./gradlew bootRun
    ```
    - Server: `http://localhost:8080`
    - API: `http://localhost:8080/api/ping`

### 2. Frontend (Next.js)
Requires Node.js 20+.

1.  **Install Dependencies:**
    ```bash
    cd frontend
    npm install
    ```
2.  **Run Dev Server:**
    ```bash
    npm run dev
    ```
    - Application: `http://localhost:3000`
